## 五、缺陷管理

### 5.1 缺陷等级

| 等级 | 定义 | 示例 |
|------|------|------|
| P0 | 系统崩溃，无法继续 | 页面无法打开、数据丢失 |
| P1 | 严重功能缺失 | 新增功能无法保存 |
| P2 | 一般缺陷 | 数据显示错误、样式问题 |
| P3 | 轻微问题 | 提示信息不准确 |

### 5.2 缺陷状态

```
新建 → 确认 → 修复中 → 已修复 → 验证通过 → 关闭
              ↓
           暂不处理
```
